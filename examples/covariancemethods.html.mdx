> Comparing different covariance estimation methods for hierarchical
> forecast reconciliation

MinTrace reconciliation minimizes the trace of the reconciled forecast
error covariance. The key input is the error covariance matrix **W**,
which can be estimated in different ways. This notebook demonstrates all
available covariance methods and compares their performance on the
Australian Domestic Tourism dataset.
